# Gerenciador de Campeonato

## Stack
- Next.js
- React
- Prisma
- SQLite

## Como Rodar Localmente
- Next.js
- React
- Prisma
- SQLite

Clone o repositório:
```bash
git clone https://github.com/seu-usuario/campeonato.git
cd campeonato
```

Instale as dependências:
```bash
npm install
```

Configure o banco de dados:
```bash
npx prisma migrate dev
```

Inicie o servidor:
```bash
npm run dev
```

Acesse http://localhost:3000

## Funcionalidades
- Cadastro de usuários
- Criação de campeonatos
- Cadastro de times por campeonato
- Agendamento de partidas entre times
